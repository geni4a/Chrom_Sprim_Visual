# Chrom_Sprim_Visual
## Packages that need to be installed
- bokeh 
- pandas
- numpy
- os

To install packages, you can use pip or pip 3
```
pip install bokeh
```
if this doesn't work, try:
```
pip install --upgrade --force-reinstall bokeh
```

Make sure the path to which packages are installed are on PATH
```
export PATH=/path/to/directory:$PATH
```

## If not using own files
- Clone repository
- Locate directory 

type 
```
  bokeh serve --show .

```


if facing issues even after installing the necessary packages, 
type 
```
  python -m bokeh serve --show .

```
  or
  ```
  python3 -m bokeh serve --show .

```
